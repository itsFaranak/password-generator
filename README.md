# Password Generator (Python + PyQt5)

A simple and beautiful desktop application built with **Python** and **PyQt5**.  
It allows you to generate strong, random passwords with adjustable length and copy them easily to your clipboard.

---

## Features
- Modern, clean user interface (built with Qt Designer)
- Adjustable password length
- Mix of uppercase, lowercase, numbers, and symbols
- One-click copy to clipboard
- Lightweight and easy to use

---

## Installation & Run

### 1. Install Python and PyQt5
Make sure Python 3.x is installed.  
Then open a terminal in the project folder and run:
